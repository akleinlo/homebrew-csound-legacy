# akleinlo Csound-legacy

Homebrew tap for Csound 6.18.1_11

## How do I install these formulae?

`brew install akleinlo/csound-legacy/<formula>`

Or `brew tap akleinlo/csound-legacy` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "akleinlo/csound-legacy"
brew "<formula>"
```

For more information, see [Homebrew Documentation](https://docs.brew.sh).
