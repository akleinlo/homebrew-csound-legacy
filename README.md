# Adriankleinlosen Csound-legacy

Homebrew tap for Csound 6.18.1_11

## How do I install these formulae?

`brew install adriankleinlosen/csound-legacy/<formula>`

Or `brew tap adriankleinlosen/csound-legacy` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "adriankleinlosen/csound-legacy"
brew "<formula>"
```
